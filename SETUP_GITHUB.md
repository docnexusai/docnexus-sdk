# GitHub repo under docnexusai org

The package is configured to use **https://github.com/docnexusai/docnexus-sdk**.

If the `docnexusai` organization already exists and the repo is empty:

```bash
git remote set-url origin https://github.com/docnexusai/docnexus-sdk.git
git push -u origin main
```

Install from GitHub:

```bash
npm install github:docnexusai/docnexus-sdk
```
